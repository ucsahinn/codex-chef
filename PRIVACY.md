# Privacy

This repository is a local setup kit. It does not collect telemetry, send data
to a maintainer service, or require a hosted backend.

Install scripts copy template files into the current user's local Codex and
Agents directories. Optional skill or MCP installation commands may download
third-party packages through the user's package manager. Those tools are subject
to their own terms and privacy policies.

Do not open issues or pull requests that include real tokens, cookies, auth
files, private keys, local databases, Codex sessions, Codex memories, logs, or
screenshots containing private data.
